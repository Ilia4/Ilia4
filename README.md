## 🚀 Projects

### [HH.ru AI Screening Bot](https://github.com/Ilia4/hh-ai-screening-bot)

AI-powered service for processing job responses from HH.ru, analyzing resumes with LLM and automating candidate communication.

**Stack:** TypeScript, NestJS, Prisma, PostgreSQL, LLM API, HH.ru API

---

### [Telegram Material Tracking Bot](https://github.com/Ilia4/telegram-material-tracking-bot)

Telegram bot for tracking material movement between departments, warehouses and production areas with saving data to Google Sheets.

**Stack:** Python, Telegram Bot API, Google Sheets API, Automation

---

### [Video Transcriber + Semantic Search](https://github.com/Ilia4/video-semantic-search)

Tool for downloading audio, transcribing it with Whisper and searching through video content using embeddings.

**Stack:** Python, Whisper, Embeddings, Vector Search

---

### [Web Parser Dashboard](https://github.com/Ilia4/web-parser-dashboard)

Backend and web interface for managing parsers, starting/stopping tasks and checking parser status.

**Stack:** TypeScript, NestJS, React, Vite, API
